# 🌍 Wanderlust

A full-stack travel web application where users can explore, create, and manage travel listings with a smooth and responsive UI.

---

## 🚀 Features

✨ User Authentication (Signup/Login)
✨ Create, Edit & Delete Listings (CRUD)
✨ Responsive UI (Mobile + Desktop)
✨ Interactive travel experience platform
✨ Clean and user-friendly interface

---

## 🛠️ Tech Stack

* ⚙️ Backend: Node.js, Express.js
* 🗄️ Database: MongoDB (Mongoose)
* 🎨 Frontend: HTML, CSS, Bootstrap, EJS
* 🔐 Authentication: Sessions & Middleware

---

## ⚙️ Installation

```bash
git clone https://github.com/sourabh1021/wanderlust.git
cd wanderlust
npm install
node app.js
```

---

## 🌐 Usage

* Sign up / Login 🔐
* Create your own travel listings 🏕️
* Edit or delete listings ✏️
* Explore travel experiences 🌄
